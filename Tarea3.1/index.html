En esta página se presentan los conceptos clave del diseño web responsivo, el funcionamiento del viewport y el uso de media queries para adaptar sitios web a diferentes dispositivos.

1. Concepto de Diseño Web Responsivo

El diseño web responsivo es una técnica que permite que los sitios se adapten automáticamente al tamaño de la pantalla del dispositivo. Esto garantiza una experiencia de usuario fluida en móviles, tablets y computadoras.

Importancia: mejora la usabilidad, optimiza el SEO y ofrece una experiencia coherente. Los usuarios actuales esperan que un sitio funcione perfectamente en cualquier plataforma.

Principios básicos: cuadrículas flexibles, imágenes escalables y uso de media queries.

2. Funcionamiento del Viewport

El viewport es el área visible de una página web dentro del navegador. Su configuración se realiza con la siguiente etiqueta: 

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Esta línea indica que el contenido debe ajustarse al ancho del dispositivo, evitando que se vea muy pequeño o se requiera desplazamiento horizontal.

3. CSS3 Media Queries

Las media queries permiten cambiar los estilos según el tamaño o la orientación de la pantalla. Se usan para reorganizar elementos y hacer que un sitio sea totalmente adaptable. 

@media (max-width: 768px) {
  body {
    background-color: #eef;
  }
}

Con esto, el fondo cambiará de color si el ancho es menor o igual a 768px, demostrando cómo se aplican estilos condicionales.
